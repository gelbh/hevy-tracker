Getting Started with Hevy Tracker

1. Create Your Spreadsheet

  • If you're viewing the template, click Extensions → Hevy Tracker → Create New Spreadsheet From Template
  • Click Create Spreadsheet in the dialog
  • Open your new spreadsheet using the link provided

2. Get Your Hevy API Key

  • Go to Hevy Developer Settings
  • Copy your API key from the settings page

3. Set Up the Tracker

  • Click Extensions → Hevy Tracker → Set Hevy API Key
  • Paste your API key and click Save
  • Initial data import will begin automatically
Note: The initial import may take a few minutes depending on your workout history
