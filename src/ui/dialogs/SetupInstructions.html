Getting Started

1

Copy Template Spreadsheet

  1. Click the button below to make a copy of the template spreadsheet: Copy Template Spreadsheet
  2. Click Make a copy in the File menu
  3. The new spreadsheet will open automatically
2

Get Your Hevy API Key

  1. Go to Hevy Developer Settings
  2. Copy your API key from the settings page
3

Set Up the Tracker

  1. In your new spreadsheet, click Extensions → Hevy Tracker → Set Hevy API Key
  2. Paste your API key and click Save
  3. Initial data import will begin automatically

Important Note About Multiple Google Accounts

If you're logged into multiple Google accounts at the same time, you may experience issues with this add-on. For the best experience:

  • Log out of all Google accounts and log back in with only the account you want to use
  • Or use an incognito/private browsing window with just one account
Note: The initial import may take a few minutes depending on your workout history
